Попробуйте Запустить Программу Несколько Раз, Ибо Здесь Всё Зависит От Генератора Случайности.
Одно Вы Сможете Увидеть В Одном Запуске, Другое Другое.

Функции:
show_winner1: Генерирует Случайную Цифру
show_winner2: Генерирует Случайную Цифру, Но Больше
show_word: Генерирует Случайное "Слово" Из Чередующихся Согласных И Гласных
show_Close: Закрывает Окно
show_phraser: Генерирует Случайную Фразу Из Заранее Подготовленных Существительных, Глаголов И Прилагательных
show_JUST: Неработающая Либо Работающая Кнопка, 50/50
show_image: Открывает Сплывающее Окно Со Случайной Картинкой
show_COUNT: Считает Нажатия, До 100
show_PRESS: Перемещает Кнопку При Нажатии В Случайное Место Окна
show_MUSIC: Проигрывает Случайный Звуковой Эффект
resetbuttonsound: Автоматический Ресет
show_MUSIC2: Проигрывает Случайную Мелодию\Песню
resetbuttonsound2: Автоматический Ресет
show_STOP: Останавливает Текущую Мелодию/Песню
show_BACK: Меняет Фон Окна На Абсолютно Случайные Цвета
def delete_history: Открывает Новое Окно С Таблицей Истории Действий.
delete_history: Очищает Базу Истории
